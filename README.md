# Save the Newts 3D

A first-person 3D game where you rescue California newts crossing a rainy road at night.

## Play

Use your flashlight to spot newts and press **E** to rescue them. Avoid cars, predators, and the cliff edge.

**Controls**
- `WASD` — Move
- `Mouse` — Look around
- `E` — Rescue newt

## Run Locally

```bash
npx serve .
```

Open `http://localhost:3000`

## Tech

Three.js · Web Audio API · Vanilla JS
